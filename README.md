## Project Documentation

This document describes the structure and contents of the **Beauty

## Project Overview

The Beauty Studio website is a **static and fully responsive** website
designed to showcase beauty services, products, image galleries, blogs,
and contact information.\
It is suitable for **college projects, presentations, and small business
websites**.

## CSS Folder

-   **bootstrap.min.css**\
    Bootstrap framework used for responsive layout and components.

-   **style.css**\
    Custom CSS file used for colors, fonts, spacing, and additional
    styling.

------------------------------------------------------------------------

## Images Folder

Contains all images used across the website, including: - Service
images\
- Product images\
- Gallery images\
- Logos and profile images

------------------------------------------------------------------------

## JavaScript Folder

-   **bootstrap.bundle.min.js**\
    Required for Bootstrap components like navbar, modal, carousel, etc.

-   **script.js**\
    Custom JavaScript for additional interactivity.

------------------------------------------------------------------------

## HTML Pages

-   **index.html** -- Home page of the Beauty Studio website\
-   **about.html** -- Information about the beauty studio\
-   **products.html** -- Displays beauty products\
-   **gallery.html** -- Image gallery section\
-   **blog.html** -- Blog and beauty tips page\
-   **contact.html** -- Contact form and contact details

------------------------------------------------------------------------

## Technologies Used

-   HTML5\
-   CSS3\
-   Bootstrap 5\
-   JavaScript

------------------------------------------------------------------------

## Conclusion

This project is ideal for **college submissions and beginner-friendly
web development practice**.\
It is easy to understand, fully static, and can be enhanced in the
future with backend technologies such as **PHP, Node.js, or databases**.
