# Beauty-Studio

Beauty Studio Website – Project Documentation
This document describes the structure and contents of the Beauty Studio website project created
using HTML, CSS, Bootstrap, and JavaScript.

Live Project : https://beauty-studio-nine.vercel.app/

Project Overview
The Beauty Studio website is a static, responsive website designed to showcase beauty services,
products, gallery images, and contact details.

Folder Structure
• css/ – Contains all CSS files (Bootstrap and custom styles).
• images/ – Contains all images used across the website.
• js/ – Contains JavaScript files.
• HTML Files – Individual pages of the website.
• README.md – Basic project information.

CSS Folder
bootstrap.min.css: Bootstrap framework for responsive design.
style.css: Custom styling for layout, colors, fonts, and responsiveness.

Images Folder
Contains service images, product images, gallery images, profile pictures, and logos used
throughout the website.

JavaScript Folder
bootstrap.bundle.min.js: Required Bootstrap JavaScript components.
script.js: Custom JavaScript for interactivity.

HTML Pages
• index.html – Home page of the Beauty Studio website.
• about.html – Information about the beauty studio.
• products.html – Displays beauty products.
• gallery.html – Image gallery section.
• blog.html – Blog or beauty tips page.
• contact.html – Contact form and details.

Technologies Used
HTML5, CSS3, Bootstrap 5

Conclusion
This project is suitable for college presentations and basic business websites. It is fully static, easy
to understand, and can be enhanced further with backend integration.
